# Airbnb NYC 2019 Exploratory Data Analysis

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on the Airbnb NYC 2019 dataset using Python. The analysis aims to uncover pricing trends, room availability patterns, neighborhood-based insights, and customer behavior using data visualization and statistical analysis.

This project demonstrates practical skills in:
- Data Cleaning
- Data Analysis
- Data Visualization
- Business Insight Generation
- Exploratory Data Analysis (EDA)

---

## Problem Statement
Airbnb listings in New York City vary significantly in pricing, room type, availability, and neighborhood popularity.

The objective of this analysis is to:
- Understand pricing distribution across listings
- Compare room types and their impact on pricing
- Analyze neighborhood-level trends
- Detect outliers and missing values
- Generate meaningful business insights from raw data

---

## Dataset Information
**Dataset Name:** Airbnb NYC 2019 Dataset

The dataset contains information such as:
- Listing ID
- Listing Name
- Host ID
- Host Name
- Neighborhood Group
- Neighborhood
- Latitude
- Longitude
- Room Type
- Price
- Minimum Nights
- Number of Reviews
- Last Review Date
- Reviews Per Month
- Calculated Host Listings Count
- Availability (365 days)

## Key Insights
Some major findings from the analysis:

- Entire homes/apartments have the highest average prices.
- Private rooms are more affordable compared to entire homes.
- Manhattan contains the highest-priced listings.
- Brooklyn has a large number of Airbnb listings.
- Price distribution contains significant outliers.
- Availability varies considerably across neighborhoods.
